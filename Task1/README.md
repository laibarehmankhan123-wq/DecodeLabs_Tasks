# DecodeLabs Project 1 – To-Do List (Python)

A simple command-line To-Do List application built using Python. This project allows users to manage their daily tasks by adding, viewing, and searching tasks through a menu-driven interface.

---

## Features

- Add new tasks
- View all saved tasks
- Search tasks by keyword
- Exit the application
- Prevents empty task entries
- Automatically assigns a unique ID to each task

---

## Technologies Used

- Python 3
- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input
- String Methods

## How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open the project folder in your terminal.
4. Run the following command:

```bash
python To_Do_List.py
```
